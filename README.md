Code used in my Cascadia R Conf 2026 presentation.

This script pulls in the BFRO data, simulates some covariates across the study area, and creates an inhomogeneous Poisson point process model.

Data is from Kaggle user mexwwell and can be found here: https://www.kaggle.com/datasets/mexwell/bigfoot-sightings 
Original data source is here: https://www.bfro.net/
